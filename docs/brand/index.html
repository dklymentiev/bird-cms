/ 03 — logo

Logo Marks

The mark is built from 39 polygonal facets in 5 anatomical groups (l-wing, r-wing, head, body, tail). 49 hand-placed vertices. Designed to read clean from 16 px favicon up to large hero sizes.

Animation rules

Intro animations play once per page load — never infinite loops. After the intro settles, the mark is static. Hover may trigger one brief replay (optional).

If you can pause it at any moment and it still reads as a logo — it's done right. If pausing breaks the impression, it's animation, not branding.

/ 05 — imagery

AI Image Prompt

Brand-consistent imagery for AI generators. Forest backdrop, polygonal accents, warm light, geometry-first composition.

Avoid
